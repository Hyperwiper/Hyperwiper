30 // When changing multiple audio object settings that must update at
31 // the same time, these functions allow the audio library interrupt
32 // to be disabled. For example, you may wish to begin playing a note
33 // in response to reading an analog sensor. If you have "velocity"
34 // information, you might start the sample playing and also adjust
35 // the gain of a mixer channel. Use AudioNoInterrupts() first, then
36 // make both changes to the 2 separate objects. Then allow the audio
37 // library to update with AudioInterrupts(). Both changes will happen
38 // at the same time, because AudioNoInterrupts() prevents any updates
39 // while you make changes.
40 //
41 #define AudioNoInterrupts() (NVIC_DISABLE_IRQ(IRQ_SOFTWARE))
42 #define AudioInterrupts() (NVIC_ENABLE_IRQ(IRQ_SOFTWARE))
